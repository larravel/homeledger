# Context

## Project
HomeLedger – Household Billing & Expense Manager

## Description
HomeLedger is a web-based system designed to help households manage recurring bills, track payments, and monitor expenses.

## Users
- Families
- Working adults
- Renters
- Parents managing finances

## Problem
Households struggle with:
- Tracking due dates
- Managing multiple bills
- Avoiding late payments

## Solution
A centralized system that:
- Tracks bills and payments
- Provides reminders
- Shows financial summaries
